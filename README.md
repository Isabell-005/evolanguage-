# 2ï¸âƒ£ Overview of German Levels (A1â€“C2)

### **A1**  
Basic everyday communication. Required for **Family Reunion Visa**.

### **A2**  
Simple communication in familiar situations.

### **B1**  
Independent use of language. Required for **Citizenship**, some **Job-Seeker Visas**.

### **B2**  
Professional communication. Required for **Skilled Worker Visa**, **Nursing**, many jobs.

### **C1**  
Academic and advanced language level. Required for German **universities** and **medical professions**.

### **C2**  
Near-native fluency. Advantage for **management**, **executive**, and high-level positions.
